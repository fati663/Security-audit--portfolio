# Q2 - Security Controls Checklist
The security controls checklist for the fictitious retail company includes:

- Antivirus installed on all employee computers
- Strong passwords enforced for all accounts
- Physical security measures in place for the store and warehouse
- Access control for employees (restricted areas)
- Surveillance cameras in store and warehouse
- Regular backups of all important data
- Security policy documentation available and followed

- This checklist will help assess the current security level and identify areas for improvement.
